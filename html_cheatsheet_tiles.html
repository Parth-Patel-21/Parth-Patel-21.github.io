Popular HTML Elements

HTML stands for HyperText Markup Language. The HyperText part refers to the fact that HTML allows you to create links that allows visitors to move from one page to another quickly and easily. A markup language allows you to annotate text, and these annotations provide additional meaning to the contents of a document. If you think of a web page, we add code around the origianl text we want to display and the browser then uses the code to display the page correctly. So the tags we add are the markup.
